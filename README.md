# Pinheiro de Campos Advogados | Template Premium

Este projeto agora inclui uma proposta de redesign institucional para a homepage da Pinheiro de
Campos Advogados, com foco em:

- posicionamento mais premium para o publico brasileiro
- comunicacao em portugues com linguagem mais atual
- destaque para direito trabalhista e direitos do caminhoneiro
- integracao visivel com o Instagram oficial
- direcao visual renovada para fotos e conteudo social

## Arquivos principais

- `frontend/index.html`: homepage completa do template
- `frontend/styles.css`: direcao visual, responsividade e animacoes
- `frontend/app.js`: revelacao suave de secoes e formulario demonstrativo
- `frontend/assets/`: artes SVG locais usadas como placeholders premium

## Como visualizar

Abra `frontend/index.html` no navegador.

## Observacao

O template e estatico e foi preparado para facilitar a proxima etapa de integracao com:

- WhatsApp oficial
- formulario real de contato
- backend / CRM
- fotos profissionais do escritorio
